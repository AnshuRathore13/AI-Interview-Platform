# AI-Interview-Platform

A full-stack AI-powered mock interview platform that generates role-specific interview questions and evaluates candidate responses in real time using LLM-based analysis.

Implemented resume parsing, speech-to-text integration, and adaptive follow-up questioning to simulate realistic technical, HR, and behavioral interview rounds.

Delivered automated feedback scoring on communication, technical accuracy, and confidence, giving users instant, structured interview prep instead of manual mock sessions
